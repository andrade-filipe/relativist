## Context

The current `src/protocol.rs` is a stub file with only a doc comment. SPEC-06 defines many types (Message, FrameHeader, ProtocolError, NodeConfig), functions (send_frame, recv_frame, connect_with_retry), and orchestration logic (coordinator loop, worker loop) that need separate sub-modules for clarity. This task converts `src/protocol.rs` into a `src/protocol/` directory with a `mod.rs` that re-exports all public items.

- [ ] `src/protocol/mod.rs` re-exports all public items from sub-modules
- [ ] `src/lib.rs` compiles with `pub mod protocol;` unchanged
- [ ] Each sub-module file is created with a doc comment placeholder

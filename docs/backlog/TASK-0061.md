## Context

SPEC-05 R37 specifies a `WorkerRoundStats` structure that captures per-worker statistics for a single round: how many agents the worker had before and after local reduction, and how many local redexes it resolved. This data is collected per round and stored in `GridMetrics.worker_stats_per_round`. It is a SHOULD-level requirement but is needed for meaningful benchmark analysis.

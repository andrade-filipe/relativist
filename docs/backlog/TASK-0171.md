## Context

This is the first and simplest net generator. `ep_annihilation(n)` produces N pairs of ERA agents connected at principal ports (SPEC-12, R37). After reduction, the net should be empty (0 agents). This is the Profile A baseline benchmark (SPEC-09, R9).

The generator function is a pure function with signature `fn(u32) -> Net` (R35), reusable by both the `generate` CLI subcommand and the benchmark suite (R36). In debug mode, the output net is validated against invariants T1-T7 (R34).

This task also establishes the module structure for `examples.rs` and the `ExampleNet` enum used by the `generate` subcommand.

```rust
use crate::net::{Net, Symbol, PortRef};

/// Available pre-built example nets (SPEC-12, R33).
#[derive(Debug, Clone, Copy, clap::ValueEnum)]
pub enum ExampleNet {
    /// N ERA-ERA annihilation pairs (Profile A). SPEC-09 R9.
    EpAnnihilation,
    /// N CON-CON annihilation pairs (Profile A). SPEC-09 R10.
    EpAnnihilationCon,
    /// N DUP-DUP annihilation pairs (Profile A). SPEC-09 R11.
    EpAnnihilationDup,
    /// N CON-DUP expansion pairs (Profile B). SPEC-09 R12.
    ConDupExpansion,
    /// Two binary trees of depth N (Profile C). SPEC-09 R13.
    DualTree,
    /// Tree sum with N work units (Profile A/B). SPEC-09 R14.
    TreeSum,
    /// Balanced tree sum with N work units. SPEC-09 R15.
    TreeSumBalanced,
    /// Mixed net with N pairs of each rule type. SPEC-09 R16.
    MixedRules,
    /// ERA propagation chain of length N (Profile C). SPEC-09 R17.
    ErasurePropagation,
    /// Church numeral encoding of N (Profile B). SPEC-14 R26.
    ChurchNat,
    /// Church addition (Profile B). SPEC-14 R26.
    ChurchAdd,
    /// Church multiplication (Profile B). SPEC-14 R26.
    ChurchMul,
}

/// Dispatch to the appropriate generator (SPEC-12, R35).
pub fn generate_example(example: ExampleNet, size: u32) -> Net {
    match example {
        ExampleNet::EpAnnihilation => ep_annihilation(size),
        // ... other variants delegate to their generators ...
        _ => todo!("generator not yet implemented"),
    }
}

/// Generate N ERA-ERA annihilation pairs (SPEC-12, R37).
///
/// Each pair: two ERA agents with principal ports connected.
/// After reduction: empty net (0 agents).
pub fn ep_annihilation(n: u32) -> Net {
    let mut net = Net::new();
    for _ in 0..n {
        let a = net.create_agent(Symbol::Era);
        let b = net.create_agent(Symbol::Era);
        net.connect(
            PortRef::AgentPort(a, 0),
            PortRef::AgentPort(b, 0),
        );
    }
    debug_assert!(/* validate invariants */);
    net
}
```

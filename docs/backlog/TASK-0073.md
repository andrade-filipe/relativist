## Context

Several places in the grid loop need to count the number of live agents in a net: `agents_per_round` recording, `WorkerRoundStats.agents_before/after`, capacity estimation. This task implements a simple helper function or method that counts `Some` entries in the agent arena. It is trivial but used frequently enough to warrant a named function.

## Acceptance Criteria

- [ ] Function `count_live_agents` is implemented (either as a free function in `src/merge.rs` or as a method on `Net`)
- [ ] Returns the count of `agents.iter().filter(|s| s.is_some()).count()`
- [ ] Complexity is O(A_capacity) where A_capacity is the length of the agents vector
- [ ] The module compiles with `cargo check`

## Files to Create/Modify

- `src/merge.rs` -- modify: add `count_live_agents` function (or `src/net/net.rs` if added as a method)

## Key Types / Signatures

```rust
/// Counts the number of live (non-None) agents in the net.
///
/// Used for metrics recording (SPEC-05, R35).
/// Complexity: O(capacity) where capacity is agents.len().
fn count_live_agents(net: &Net) -> usize {
    net.agents.iter().filter(|slot| slot.is_some()).count()
}
```

## Test Expectations

- Unit test: empty net -> 0
- Unit test: net with 3 agents -> 3
- Unit test: net with holes (some slots None) -> correct count

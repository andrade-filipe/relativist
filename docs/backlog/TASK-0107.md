## Context

SPEC-13 R19 defines the coordinator's finite state machine with 9 states: Init, WaitingForWorkers, Partitioning, Dispatching, WaitingForResults, Merging, CheckTermination, Done, Error. The FSM follows the stimulus-response pattern (R20): a pure function `transition(state, event) -> (new_state, Vec<Action>)`. This task defines the state enum, the event enum, and the action enum. The actual transition logic is in TASK-0108.

The FSM is enum-based (R22), not typestate, for serialization and logging ergonomics.

## Key Types / Signatures

```rust
//! Coordinator FSM for distributed IC reduction.
//!
//! The coordinator implements the BSP (Bulk Synchronous Parallel) programming
//! model (SPEC-13 R1, R4). Each grid round is one BSP superstep.

use crate::net::Net;
use crate::partition::Partition;
use crate::protocol::Message;
use crate::error::CoordinatorError;
use std::net::SocketAddr;
use std::time::Duration;

/// Unique identifier for a connected worker.
pub type WorkerId = u32;

/// Unique identifier for a timer.
pub type TimerId = u32;

/// Coordinator FSM states (SPEC-13 R19).
#[derive(Debug, Clone, PartialEq, Eq, serde::Serialize)]
pub enum CoordinatorState {
    Init,
    WaitingForWorkers,
    Partitioning,
    Dispatching,
    WaitingForResults,
    Merging,
    CheckTermination,
    Done,
    Error,
}

/// Events that drive the coordinator FSM (SPEC-13 R20).
#[derive(Debug, Clone)]
pub enum CoordinatorEvent {
    ConfigLoaded,
    WorkerRegistered(WorkerId),
    SplitComplete(Vec<Partition>),
    AllDispatched,
    PartitionReturned { worker_id: WorkerId, partition: Partition },
    HeartbeatTimeout(WorkerId),
    MergeComplete { net: Net, border_redex_count: usize },
    FatalError(String),
}

/// Actions the coordinator runtime must execute (SPEC-13 R20).
#[derive(Debug)]
pub enum CoordinatorAction {
    BindListener(SocketAddr),
    SendMessage(WorkerId, Message),
    StartTimer(TimerId, Duration),
    CancelTimer(TimerId),
    LogTransition { from: CoordinatorState, to: CoordinatorState },
    WriteOutput(Net),
    ShutdownAll,
}
```

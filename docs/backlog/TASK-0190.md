## Context

SPEC-09 Section 3.2.1 (R11) defines the **EP-Annihilation-DUP** benchmark: N pairs of DUP agents connected by principal ports, with auxiliary ports connected to FreePorts. This is Profile A (embarrassingly parallel) and exercises the DUP-DUP annihilation rule, which produces a parallel reconnection pattern.

Complements EP-Annihilation-CON (TASK-0189). The asymmetry between CON-CON (cross) and DUP-DUP (parallel) reconnection is essential for the system's universality (REF-002, p. 90).

After reduction: 0 DUP agents remain. FreePorts are reconnected in the parallel pattern (a.1<->b.1, a.2<->b.2 for each pair).

```rust
/// EP-Annihilation-DUP: N pairs of DUP agents connected by principal ports.
/// Auxiliary ports connected to FreePorts.
/// Profile A (embarrassingly parallel). All redexes independent.
/// Rules exercised: DUP-DUP (annihilation parallel-reconnect).
/// Expected result: 0 DUP agents, FreePorts parallel-connected.
///
/// SPEC-09 R11. Fills AC-005 Gap L6.
pub struct EPAnnihilationDup;

impl Benchmark for EPAnnihilationDup {
    fn id(&self) -> BenchmarkId {
        BenchmarkId::EPAnnihilationDup
    }

    fn describe(&self, size: u32) -> String {
        format!("EP-Annihilation (DUP): {} pairs, {} agents, {} redexes",
                size, size * 2, size)
    }

    fn make_net(&self, n: u32) -> Net {
        let mut net = Net::with_capacity((n * 2) as usize);
        for i in 0..n {
            let a = net.create_agent(Symbol::Dup);
            let b = net.create_agent(Symbol::Dup);
            // Principal ports: DUP.0 <-> DUP.0 (forms a redex)
            net.connect(PortRef::AgentPort(a, 0), PortRef::AgentPort(b, 0));
            // Auxiliary ports -> FreePorts
            net.connect(PortRef::AgentPort(a, 1), PortRef::FreePort(4 * i));
            net.connect(PortRef::AgentPort(a, 2), PortRef::FreePort(4 * i + 1));
            net.connect(PortRef::AgentPort(b, 1), PortRef::FreePort(4 * i + 2));
            net.connect(PortRef::AgentPort(b, 2), PortRef::FreePort(4 * i + 3));
        }
        net
    }

    fn default_sizes(&self) -> Vec<u32> {
        vec![100, 500, 1_000, 5_000, 10_000, 50_000]
    }

    fn verify(&self, _sequential_result: &Net, distributed_result: &Net) -> bool {
        count_live_agents(distributed_result) == 0
    }
}
```

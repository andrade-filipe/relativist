## Context

The arity of an agent determines how many auxiliary ports it has. CON and DUP have arity 2 (2 auxiliary ports); ERA has arity 0 (no auxiliary ports). The total number of ports is `arity + 1` (including the principal port at index 0). These functions are used throughout the system: in `remove_agent` (to know how many ports to disconnect), in debug assertions (to validate port indices), and in the reduction rules (to iterate over auxiliary ports).

- [ ] `total_ports(Symbol) -> u8` is a `pub const fn` that returns `arity(symbol) + 1`
- [ ] Both functions are publicly exported from the `net` module

## Key Types / Signatures

```rust
/// Returns the arity (number of auxiliary ports) of a symbol.
/// CON and DUP: 2 auxiliary ports. ERA: 0 auxiliary ports.
pub const fn arity(symbol: Symbol) -> u8 {
    match symbol {
        Symbol::Con => 2,
        Symbol::Dup => 2,
        Symbol::Era => 0,
    }
}

/// Returns the total number of ports of a symbol (arity + 1).
/// Includes the principal port (port 0).
pub const fn total_ports(symbol: Symbol) -> u8 {
    arity(symbol) + 1
}
```

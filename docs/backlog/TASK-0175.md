## Context

`mixed_rules(n)` produces a net containing N pairs of each of the 6 interaction rule types, for a total of 6N initial redex pairs (R41). The 6 rule types are: CON-CON annihilation, DUP-DUP annihilation, ERA-ERA void, CON-DUP commutation, CON-ERA erasure, and DUP-ERA erasure.

This generator tests that the reduction engine handles all rule types correctly in a single net. It combines patterns from the simpler generators (ERA-ERA from TASK-0171, CON-CON/DUP-DUP from TASK-0172, CON-DUP from TASK-0173) with the addition of CON-ERA and DUP-ERA erasure pairs.

SPEC-09 R16 defines this as the mixed-rules benchmark.

## Acceptance Criteria

- [ ] `mixed_rules(n: u32) -> Net` generates a net with 6 rule types, N pairs each
- [ ] N ERA-ERA pairs (void rule): 2 ERA agents, principals connected
- [ ] N CON-CON pairs (annihilation): 2 CON agents, principals connected, auxiliaries to free ports
- [ ] N DUP-DUP pairs (annihilation): 2 DUP agents, principals connected, auxiliaries to free ports
- [ ] N CON-DUP pairs (commutation): 1 CON + 1 DUP, principals connected, auxiliaries to free ports
- [ ] N CON-ERA pairs (erasure): 1 CON + 1 ERA, principals connected, CON auxiliaries to free ports
- [ ] N DUP-ERA pairs (erasure): 1 DUP + 1 ERA, principals connected, DUP auxiliaries to free ports
- [ ] Total initial redexes: `6 * n`
- [ ] Free port IDs are sequential and non-overlapping across all pair types
- [ ] Output validates in debug mode (R34)
- [ ] `generate_example` dispatch is updated for the `MixedRules` variant

/// Rule types: ERA-ERA, CON-CON, DUP-DUP, CON-DUP, CON-ERA, DUP-ERA.

## Notes

- Agent count per pair type:
  - ERA-ERA: 2 ERA (0 free ports)
  - CON-CON: 2 CON (4 free ports)
  - DUP-DUP: 2 DUP (4 free ports)
  - CON-DUP: 1 CON + 1 DUP (4 free ports)
  - CON-ERA: 1 CON + 1 ERA (2 free ports for CON auxiliaries)
  - DUP-ERA: 1 DUP + 1 ERA (2 free ports for DUP auxiliaries)
- Total agents per iteration: 2 + 2 + 2 + 2 + 2 + 2 = 12. Total = 12n.
- Total free ports per iteration: 0 + 4 + 4 + 4 + 2 + 2 = 16. Total = 16n.
- For erasure pairs (CON-ERA, DUP-ERA): the ERA agent has only a principal port. The non-ERA agent's auxiliaries connect to free ports.

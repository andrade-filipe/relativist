## Context

The Fundamental Property of Relativist (SPEC-01, G1) states: `reduce_all(net) ~ extract_result(run_grid(net, n))`. This is the single most important invariant of the entire system. This task creates integration tests that validate G1 by comparing the Normal Form obtained via sequential reduction against the Normal Form obtained via the grid loop, for various net topologies and worker counts.

This is the empirical validation of the TCC's central hypothesis.

## Acceptance Criteria

- [ ] Test file `tests/fundamental_property.rs` (or equivalent) is created
- [ ] For each test net:
  - Clones the net
  - Runs `reduce_all` on clone A -> Normal Form A
  - Runs `run_grid` on clone B -> Normal Form B
  - Asserts Normal Form A is isomorphic to Normal Form B
  - Asserts total interaction counts are equal (SPEC-01, T7)
- [ ] Test nets include:
  - Single CON-CON annihilation (simplest case)
  - Single DUP-DUP annihilation
  - Single ERA-ERA void
  - CON-DUP commutation (creates 4 new agents)
  - CON-ERA / DUP-ERA erasure
  - Chain of 4+ agents requiring multiple reduction steps
  - Net requiring multiple grid rounds (border redexes that generate new redexes)
- [ ] Worker counts tested: 2, 4 (at minimum)
- [ ] All tests pass with `cargo test`

## Key Types / Signatures

```rust
#[test]
fn fundamental_property_con_dup_commutation() {
    let net_seq = build_con_dup_test_net();
    let net_grid = net_seq.clone();

    // Sequential reduction
    let mut seq = net_seq;
    let seq_interactions = reduce_all(&mut seq);

    // Grid reduction with 2 workers
    let config = GridConfig { num_workers: 2, max_rounds: None };
    let (grid_result, metrics) = run_grid(net_grid, &config, &ContiguousIdStrategy);

    // Fundamental Property G1
    assert_isomorphic(&seq, &grid_result);
    // Interaction count invariance T7
    assert_eq!(seq_interactions as u64, metrics.total_interactions);
}
```

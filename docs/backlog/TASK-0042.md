## Context

`PartitionPlan` is the top-level output of the `split` function. It holds all partitions (one per worker) and the border map recording the original endpoints of every wire that was cut during partitioning. The border map is essential for the merge protocol (SPEC-05) to restore connections. This struct is the data contract between the split operation and the merge operation.

## Key Types / Signatures

```rust
use std::collections::HashMap;

/// The complete partitioning plan.
#[derive(Debug, Clone)]
pub struct PartitionPlan {
    /// List of partitions, one per worker. partitions[i].worker_id == i.
    pub partitions: Vec<Partition>,

    /// Border map: borderId -> (original_endpoint_A, original_endpoint_B).
    /// Records the two endpoints of each wire that was cut.
    /// Used by the merge protocol (SPEC-05) to restore connections.
    pub borders: HashMap<u32, (PortRef, PortRef)>,
}
```

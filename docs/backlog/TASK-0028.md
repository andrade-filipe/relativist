## Context

SPEC-03 Section 4.6.2 defines `reduce_all`, which repeatedly calls `reduce_step` until the net reaches Normal Form (empty redex queue). It returns a `ReductionStats` struct that tracks the total number of interactions and a breakdown by rule type (annihilation, commutation, erasure, void). This per-type counting satisfies R17 (discriminate by rule type for workload profiling).

`reduce_all` is the primary entry point for sequential reduction. WARNING: it does not terminate if the net is non-terminating. For safety, use `reduce_n` with a budget.

Complexity: O(S) where S is the total number of interactions to Normal Form (invariant T7 from SPEC-01).

```rust
/// Statistics of a completed reduction (SPEC-03 Section 4.6.2, R17).
///
/// Tracks total interactions and a breakdown by rule type for workload
/// profiling. The per-type counts satisfy R17 (discriminate by rule type).
#[derive(Debug, Clone, Default, PartialEq, Eq)]
pub struct ReductionStats {
    /// Total number of interactions performed.
    pub total_interactions: u64,
    /// Number of annihilation rules applied (CON-CON + DUP-DUP).
    pub anni_count: u64,
    /// Number of commutation rules applied (CON-DUP).
    pub comm_count: u64,
    /// Number of erasure rules applied (CON-ERA + DUP-ERA).
    pub eras_count: u64,
    /// Number of void rules applied (ERA-ERA).
    pub void_count: u64,
}

impl ReductionStats {
    /// Records one interaction of the given rule type.
    #[inline]
    pub fn record(&mut self, rule: Rule) {
        self.total_interactions += 1;
        match rule {
            Rule::Anni => self.anni_count += 1,
            Rule::Comm => self.comm_count += 1,
            Rule::Eras => self.eras_count += 1,
            Rule::Void => self.void_count += 1,
        }
    }
}

/// Reduces the net to Normal Form (SPEC-03 Section 4.6.2, R13).
///
/// WARNING: does not terminate if the net is non-terminating.
/// For potentially non-terminating nets, use reduce_n.
///
/// Complexity: O(S) where S is the total number of interactions to Normal Form
/// (invariant T7 from SPEC-01 guarantees S is unique for the given net).
///
/// Returns reduction statistics with per-rule-type breakdown.
pub fn reduce_all(net: &mut Net) -> ReductionStats {
    let mut stats = ReductionStats::default();

    loop {
        match reduce_step(net) {
            StepResult::NormalForm => return stats,
            StepResult::Reduced(rule) => stats.record(rule),
        }
    }
}
```

## Notes

- The `record` helper method on `ReductionStats` avoids duplicating the match-on-rule logic in both `reduce_all` and `reduce_n`.
- `ReductionStats` derives `Default` so it can be initialized with all zeros without listing every field.
- The total interaction count is deterministic for any terminating net (T7): running `reduce_all` with any redex ordering produces the same total. This is a key property for the TCC's validation.

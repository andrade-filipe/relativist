## Description

Implement `create_transport()` factory function in `src/protocol/transport.rs` (or a dedicated file):

```rust
pub fn create_transport(
    bind_addr: SocketAddr,
    config: &TransportConfig,
) -> Result<Box<dyn Transport>, ProtocolError>
```

Behavior:
- `TransportBackend::Tcp` → `Box::new(TcpTransport::new(bind_addr, config.clone()))`
- `TransportBackend::Unix` on `cfg(unix)` → `Box::new(UnixTransport::new(path))`
- `TransportBackend::Unix` on `cfg(not(unix))` → `Err("not supported on this platform")`
- `TransportBackend::Channel` → `Err("use ChannelTransport::pair() directly")`

Default UDS path: `/tmp/relativist.sock` when `unix_socket_path` is `None`.

**Phase:** v2 — Milestone M1
**Priority:** P1
**Complexity:** S (~30 LoC)
**Depends on:** TASK-0308, TASK-0309, TASK-0310

## Description

1. **Same-host detection** (R33-R34): When coordinator's bind address is `127.0.0.1` or `::1` and backend is `Tcp`, log at `info` level recommending `--transport=unix`. MUST NOT auto-switch (R34).

4. Full end-to-end: CLI → config → transport → coordinator/worker
5. Integration tests: IT1 (full grid cycle via TcpTransport)
6. All 690+ tests pass (including new transport tests)

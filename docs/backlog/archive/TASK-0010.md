## Context

`get_target` and `set_port` are the low-level port array accessors. `get_target(port) -> PortRef` reads what a port is connected to by looking up the port array at the computed index. `set_port(port, target)` writes a target into the port array. Both operate only on `AgentPort` references (since `FreePort` has no slot in the port array).

These are the building blocks for `connect`, `disconnect`, and the debug assertions. `get_target` is public (R15); `set_port` is a private helper.

## Acceptance Criteria

- [ ] `get_target(port: PortRef) -> PortRef` is a `pub` method on `Net`
- [ ] For `AgentPort(id, p)`, returns `ports[port_index(id, p)]` if in bounds, else `DISCONNECTED`
- [ ] For `FreePort(_)`, returns `DISCONNECTED` (FreePort targets are resolved during merge)
- [ ] `set_port(port: PortRef, target: PortRef)` is a private (`fn`, not `pub fn`) method on `Net`
- [ ] For `AgentPort(id, p)`, writes `target` to `ports[port_index(id, p)]` if in bounds
- [ ] For `FreePort(_)`, `set_port` is a no-op (FreePort has no slot)
- [ ] Both functions have O(1) complexity

## Key Types / Signatures

```rust
impl Net {
    /// Returns the PortRef to which the given port is connected.
    ///
    /// Complexity: O(1).
    pub fn get_target(&self, port: PortRef) -> PortRef {
        match port {
            PortRef::AgentPort(id, p) => {
                let idx = port_index(id, p);
                if idx < self.ports.len() {
                    self.ports[idx]
                } else {
                    DISCONNECTED
                }
            }
            PortRef::FreePort(_) => {
                // FreePort targets are resolved during merge (SPEC-05).
                DISCONNECTED
            }
        }
    }

    /// Writes the target of a port in the port array.
    /// Only operates on AgentPort; FreePort is ignored.
    fn set_port(&mut self, port: PortRef, target: PortRef) {
        if let PortRef::AgentPort(id, p) = port {
            let idx = port_index(id, p);
            if idx < self.ports.len() {
                self.ports[idx] = target;
            }
        }
    }
}
```

## Notes

- `set_port` is intentionally private (`fn`, not `pub fn`). External code should use `connect` and `disconnect` to maintain bidirectionality (invariant T1/I1).
- The bounds check (`idx < self.ports.len()`) is defensive. In normal operation, the port array is always large enough for all live agents (ensured by `create_agent`).

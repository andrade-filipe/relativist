## Context

`build_add(a: u64, b: u64) -> Net` constructs an IC net that, when reduced to Normal Form via `reduce_all`, yields `church(a + b)`. The function encodes `add = lambda m. lambda n. lambda f. lambda x. m f (n f x)`.

Per R4b and SC-007, the function constructs everything in a single `Net` by calling `encode_church_into` sequentially for both operands. Agent IDs are assigned monotonically from the same `next_id` counter, satisfying I3 (SPEC-01).

The addition combinator produces O(a + b) agents and O(a + b) reduction interactions (R20). This is Profile A/B behavior.

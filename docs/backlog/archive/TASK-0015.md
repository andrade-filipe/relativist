## Context

In debug mode (`#[cfg(debug_assertions)]`), the net must verify its representation invariants after each mutation. These assertions catch bugs early during development and testing by validating:

- **I1 (Bidirectional port array, R18):** If `ports[a] == b`, then `ports[b] == a`. Also checks T1 constraints: no port-to-self connections, DISCONNECTED only at root port, cross-port self-loops accepted. This implements the linearity invariant T1.
- **I2 (Reference validity, R19):** Every `AgentPort(id, p)` reference in the port array points to an existing agent with `p <= arity(agent.symbol)`.
- **I3 (Monotonic IDs):** `next_id` is strictly greater than any `AgentId` in use.
- **I6 (ERA Auxiliary Slot Cleanliness, SPEC-01 Revised v3):** ERA agents' unused auxiliary slots (indices 1 and 2) MUST contain `DISCONNECTED` and MUST NOT be referenced by any port. Complements T1/I1 for unused port array slots.
- **I7 / R6a (Root Port Consistency):** If `net.root` is `Some(ref)`, it MUST be `AgentPort(id, 0)` where `id` is a live agent (R6a). FreePort roots are NOT valid (R6a). The root port's slot in the port array MUST contain `DISCONNECTED` (R18a).

Additionally, a `count_stale_redexes` diagnostic function and a convenience `assert_all_invariants` that calls all checks.

In release mode (R21), these assertions are compiled out for performance.

## Acceptance Criteria

- [ ] `assert_adjacency_consistent()` verifies I1/T1 for all live agents, including root port exception (DISCONNECTED allowed at root port), port-to-self rejection, and cross-port self-loop acceptance; panics with a descriptive message on violation
- [ ] `assert_refs_valid()` verifies I2 for all live agents, panics with a descriptive message on violation
- [ ] `assert_next_id_valid()` verifies I3 for all agent slots, panics with a descriptive message on violation
- [ ] `assert_era_unused_ports_clean()` verifies ERA agents' unused port slots (ports 1 and 2) contain DISCONNECTED (SPEC-02 v3, SC-007); panics with a descriptive message on violation
- [ ] `assert_root_consistent()` verifies R6a + I7: root MUST be `None` or `Some(AgentPort(id, 0))` where `id` is a live agent; FreePort roots MUST be rejected (R6a); root port slot MUST contain DISCONNECTED (R18a)
- [ ] `count_stale_redexes() -> usize` counts stale entries in the redex queue
- [ ] `assert_all_invariants()` calls all five assertion functions (I1, I2, I3, I6, I7)
- [ ] All functions are gated by `#[cfg(debug_assertions)]`
- [ ] In release mode, none of these functions exist (compile-time elimination)

```rust
#[cfg(debug_assertions)]
impl Net {
    /// Verifies I1/T1: bidirectionality of the port array.
    /// Also checks T1 constraints: no port-to-self, cross-port self-loops
    /// accepted.
    ///
    /// R18a: The root agent's principal port is explicitly exempt from T1.
    /// The function computes the root agent ID and skips ONLY that specific
    /// port, rather than generically skipping all DISCONNECTED ports.
    pub fn assert_adjacency_consistent(&self) {
        // R18a: determine root agent ID (if any) for the exemption
        let root_agent_id: Option<AgentId> = match self.root {
            Some(PortRef::AgentPort(id, 0)) => Some(id),
            _ => None,
        };

        for slot in self.agents.iter() {
            if let Some(agent) = slot {
                let num_ports = total_ports(agent.symbol);
                for p in 0..num_ports {
                    let port = PortRef::AgentPort(agent.id, p);
                    let target = self.get_target(port);

                    // R18a: root agent's principal port is exempt.
                    if p == 0 && root_agent_id == Some(agent.id) {
                        if target == DISCONNECTED {
                            continue; // Permanent DISCONNECTED for root (R18a)
                        }
                    }

                    if target == DISCONNECTED {
                        panic!("T1 violated: port {:?} is DISCONNECTED but is not the root agent's principal port", port);
                    }

                    // Bidirectional check: ports[target] must point back to port
                    let reverse = self.get_target(target);
                    assert_eq!(reverse, port,
                        "I1 violated: {:?} -> {:?}, but {:?} -> {:?}",
                        port, target, target, reverse);
                }
            }
        }
    }

    /// Verifies I2: reference validity.
    pub fn assert_refs_valid(&self) { /* ... */ }

    /// Verifies I3: next_id > max(IDs in use).
    pub fn assert_next_id_valid(&self) { /* ... */ }

    /// Verifies ERA agents' unused port slots (ports 1 and 2) contain
    /// DISCONNECTED (SPEC-02 v3, SC-007).
    pub fn assert_era_unused_ports_clean(&self) {
        for slot in self.agents.iter() {
            if let Some(agent) = slot {
                if agent.symbol == Symbol::ERA {
                    let slot1 = agent_port_index(agent.id, 1);
                    let slot2 = agent_port_index(agent.id, 2);
                    if slot1 < self.ports.len() {
                        assert_eq!(self.ports[slot1], DISCONNECTED,
                            "I6 violated: ERA agent {} has non-DISCONNECTED value at auxiliary slot 1",
                            agent.id);
                    }
                    if slot2 < self.ports.len() {
                        assert_eq!(self.ports[slot2], DISCONNECTED,
                            "I6 violated: ERA agent {} has non-DISCONNECTED value at auxiliary slot 2",
                            agent.id);
                    }
                }
            }
        }
    }

    /// Verifies R6a + I7: root port consistency.
    /// Root MUST be None or Some(AgentPort(id, 0)) where id is a live agent.
    /// FreePort roots are rejected (R6a). The root port slot MUST contain
    /// DISCONNECTED (R18a).
    pub fn assert_root_consistent(&self) {
        if let Some(root_ref) = self.root {
            match root_ref {
                PortRef::AgentPort(id, p) => {
                    // R6a: root MUST be principal port (port 0)
                    assert_eq!(
                        p, 0,
                        "R6a violated: root must be AgentPort(id, 0), got port {}",
                        p
                    );
                    assert!(
                        self.agents.get(id as usize)
                            .is_some_and(|s| s.is_some()),
                        "R6a violated: root references nonexistent agent {}",
                        id
                    );
                    let port_idx = agent_port_index(id, 0);
                    assert_eq!(
                        self.ports[port_idx], DISCONNECTED,
                        "R18a violated: root port {:?} is not DISCONNECTED (internally connected)",
                        root_ref
                    );
                }
                PortRef::FreePort(_) => {
                    panic!("R6a violated: root is FreePort, but root MUST be None or Some(AgentPort(id, 0))");
                }
            }
        }
    }

    /// Counts stale redexes in the queue (diagnostic).
    pub fn count_stale_redexes(&self) -> usize { /* ... */ }

    /// Runs all invariant checks (I1, I2, I3, I6, I7).
    pub fn assert_all_invariants(&self) {
        self.assert_adjacency_consistent();
        self.assert_refs_valid();
        self.assert_next_id_valid();
        self.assert_era_unused_ports_clean();
        self.assert_root_consistent();
    }
}
```

## Notes

- The `#[cfg(debug_assertions)]` gate means these functions only exist in debug/test builds. In release builds (`cargo build --release`), the entire `impl` block is removed at compile time, adding zero overhead (R21).
- The `debug.rs` file needs to import types from `super::types` and use `super::net::Net`.
- These assertions are called by the reduction engine (SPEC-03) after each rule application. Getting them right here prevents bugs from propagating downstream.

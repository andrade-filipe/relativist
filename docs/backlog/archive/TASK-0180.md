## Context

SPEC-09 Section 4.6 defines the directory structure for the benchmark module. This task creates the module scaffolding under `src/bench/` with the appropriate `mod.rs` files and sub-module declarations. The bench module is a library module (not a standalone crate) so that both the CLI binary (`src/bin/bench.rs`) and criterion benchmarks (`benches/`) can import it.

This task only creates the module files with placeholder contents and re-exports. Actual types and logic are defined in subsequent tasks.

**Scope:** Migrate the entire codebase from `bincode = "1"` to `bincode = "2"`.
Mechanical refactor with semantic neutrality: bincode v2's `config::standard()`
uses varint by default, so encoded sizes will shrink but decoded values are
identical. No new types, no protocol-version bump (deferred to TASK-0347).

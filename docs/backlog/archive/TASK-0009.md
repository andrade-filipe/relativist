## Context

`create_agent` is the factory method for adding new agents to the net. It assigns the next available `AgentId` (via the bump counter `next_id`), inserts the agent into the arena, expands the port array to accommodate the new agent's slots, and returns the assigned ID. All new port slots are initialized to `DISCONNECTED`. The caller is responsible for connecting the agent's ports using `connect()`.

This operation satisfies R11 (create_agent) and R10 (next_id monotonicity). Expected complexity is O(1) amortized (may trigger Vec reallocation).

## Key Types / Signatures

```rust
impl Net {
    /// Creates a new agent with the given symbol.
    /// Returns the assigned AgentId.
    ///
    /// Complexity: O(1) amortized (may trigger Vec reallocation).
    /// Postcondition: agents[id] == Some(Agent { symbol, id }), next_id == id + 1.
    pub fn create_agent(&mut self, symbol: Symbol) -> AgentId {
        let id = self.next_id;
        self.next_id += 1;

        let agent = Agent { symbol, id };

        // Expand arena to contain index id
        if self.agents.len() <= id as usize {
            self.agents.resize((id as usize) + 1, None);
        }
        self.agents[id as usize] = Some(agent);

        // Expand port array for the new agent's ports
        let required_len = (id as usize + 1) * PORTS_PER_SLOT;
        if self.ports.len() < required_len {
            self.ports.resize(required_len, DISCONNECTED);
        }

        id
    }
}
```

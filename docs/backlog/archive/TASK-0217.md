## Context

SPEC-08 v3 (SC-017) added a dedicated test for SPEC-01 D2 (Local Reduction Equivalence). D2 states that reducing a redex inside a partition produces the same topological change as reducing that same redex in the global net. While D2 is already tested indirectly by PB10 (partition/merge roundtrip with local reduce) and all INT tests (full pipeline), this dedicated test isolates D2 from the full pipeline, improving diagnosis precision when D2-specific bugs occur.

This is a SHOULD requirement (R44), not MUST.

```rust
/// D2-1: Dedicated test for Local Reduction Equivalence (SPEC-01 D2).
///
/// Verifies that reducing a single redex in a partition produces
/// the same topological change as reducing it in the global net.
#[test]
fn test_d2_local_reduction_equivalence() {
    // For each rule type: CON-CON, CON-DUP, CON-ERA
    for fixture in [annihilation_fixture, commutation_fixture, erasure_fixture] {
        let net = fixture();
        let redex = net.redex_queue()[0]; // Known single redex

        // Global reduction: record delta
        let mut global = net.clone();
        reduce_step(&mut global);
        let global_delta = compute_delta(&net, &global);

        // Partition so redex is internal
        let plan = partition(&net, 2); // 2 workers, redex internal to worker 0
        let partition = &plan.partitions[0];

        // Local reduction in partition
        let mut local = partition.clone();
        reduce_step(&mut local);
        let local_delta = compute_delta(partition, &local);

        // Compare deltas
        assert_eq!(global_delta.agents_removed, local_delta.agents_removed);
        assert_eq!(global_delta.agents_created, local_delta.agents_created);
        // Connection changes must be equivalent modulo ID renaming
    }
}
```

## Notes

- This is a SHOULD requirement (R44). D2 is already covered indirectly by PB10 and INT1-INT11. The dedicated test adds diagnosis precision but is not critical for correctness coverage.
- The main challenge is constructing nets where the redex is guaranteed to be internal (not on a border). This requires careful fixture design or a partitioning strategy that places both agents of the redex in the same partition.
- The "topological delta" comparison is the novel aspect: it compares the structural changes (not final states) between global and local reduction of the same redex.
- Added by SPEC-08 v3 SC-017.

# TASK-0335: LambdaCodec edge cases (T5-T9 from SPEC-27 §6.3)

**Spec:** SPEC-27 R16, T5-T9
**Phase:** v2 — Layer 1 (Encoder/Decoder API), Phase 3 (LambdaCodec)
**Priority:** P0
**Complexity:** M (~120 LoC tests)
**Depends on:** TASK-0334 (decoder complete)

## Description

Add the 5 mandatory edge-case round-trip tests for `LambdaCodec` from
SPEC-27 §6.3, plus 3 negative tests, all inline in `codec_lambda.rs`
under `#[cfg(test)] mod tests`.

### T5 — Identity
```rust
let codec = LambdaCodec::new();
let out = codec.encode(br#"{"term":"λx. x"}"#).unwrap();
// Identity is already in normal form — decode without reducing
let term = codec.decode(&out).unwrap();
assert_eq!(term["term"].as_str().unwrap(), "λx. x");
```

### T6 — Single beta-reduction
```rust
let mut net = codec.encode(br#"{"term":"(λx. x) (λy. y)"}"#).unwrap();
reduce_all(&mut net);
let term = codec.decode(&net).unwrap();
// Alpha-equivalent to "λy. y" or "λx. x" depending on fresh-name policy
assert!(matches!(term["term"].as_str().unwrap(), "λx_0. x_0" | "λy. y"));
```

### T7 — Nested application (double identity)
```rust
let mut net = codec.encode(br#"{"term":"(λf. λx. f (f x)) (λy. y)"}"#).unwrap();
reduce_all(&mut net);
let term = codec.decode(&net).unwrap();
// Equivalent to identity
assert_alpha_equiv(term["term"].as_str().unwrap(), "λx. x");
```

### T8 — Erasure
```rust
let mut net = codec.encode(br#"{"term":"(λx. λy. y) (λz. z)"}"#).unwrap();
reduce_all(&mut net);
let term = codec.decode(&net).unwrap();
assert_alpha_equiv(term["term"].as_str().unwrap(), "λy. y");
// Verify ERA was used: count_live_agents includes Era during encode
```

### T9 — Duplication
```rust
let mut net = codec.encode(br#"{"term":"(λx. x x) (λy. y)"}"#).unwrap();
reduce_all(&mut net);
let term = codec.decode(&net).unwrap();
// Self-application of identity reduces to identity (alpha-equivalent)
assert_alpha_equiv(term["term"].as_str().unwrap(), "λy. y");
```

### Negative tests

- N1: malformed input `b"{not json}"` → `EncodeError::InvalidInput`
- N2: invalid term `b"{\"term\":\"(λx.\"}"` (missing body) →
  `EncodeError::InvalidInput("parse: ...")`
- N3: `decode` on a net with one disconnected stray ERA agent →
  `DecodeError::UnrecognizedStructure(...)`

## Context

SPEC-09 Section 3.2.1 (R9) defines the **EP-Annihilation** benchmark: N pairs of ERA agents connected by principal ports. All redexes are independent (embarrassingly parallel, Profile A). Expected result after reduction: 0 agents. This is the simplest benchmark and the direct mapping of AC-005 `mkEPNet`.

The net generator creates 2N agents (all ERA) with N redexes. After reduction, all agents annihilate via the ERA-ERA (void) rule. Verification checks that both sequential and distributed results have 0 live agents.

Default sizes include the Haskell prototype's sizes (R43): 100, 500, 1000, plus larger sizes up to 100K.

## Key Types / Signatures

```rust
use crate::bench::{Benchmark, BenchmarkId};
use crate::net::{Net, Symbol, PortRef};

/// EP-Annihilation: N pairs of ERA agents connected by principal ports.
/// Profile A (embarrassingly parallel). All redexes independent.
/// Rules exercised: ERA-ERA (void).
/// Expected result: 0 agents.
///
/// SPEC-09 R9. Direct mapping from AC-005 `mkEPNet`.
pub struct EPAnnihilation;

impl Benchmark for EPAnnihilation {
    fn id(&self) -> BenchmarkId {
        BenchmarkId::EPAnnihilation
    }

    fn describe(&self, size: u32) -> String {
        format!("EP-Annihilation (ERA): {} pairs, {} agents, {} redexes",
                size, size * 2, size)
    }

    fn make_net(&self, n: u32) -> Net {
        let mut net = Net::with_capacity((n * 2) as usize);
        for _ in 0..n {
            let a = net.create_agent(Symbol::Era);
            let b = net.create_agent(Symbol::Era);
            // Connect principal ports: ERA.0 <-> ERA.0 (forms a redex)
            net.connect(PortRef::AgentPort(a, 0), PortRef::AgentPort(b, 0));
        }
        net
    }

    fn default_sizes(&self) -> Vec<u32> {
        vec![100, 500, 1_000, 5_000, 10_000, 50_000, 100_000]
    }

    fn verify(&self, _sequential_result: &Net, distributed_result: &Net) -> bool {
        // Both should reduce to empty net (0 live agents).
        // ERA-ERA annihilation destroys both agents with no residual.
        count_live_agents(distributed_result) == 0
    }
}
```

## Notes

- ERA agents have only 1 port (port 0, the principal port). No auxiliary ports to connect.
- All redexes are independent: reducing any pair does not affect any other pair. This is why the benchmark is Profile A (embarrassingly parallel) with expected 1 grid round.
- The Haskell prototype reports super-linear speedup for this benchmark due to the O(N^2) `findRedexes`. Relativist should show linear speedup due to the O(1) redex queue.

## Context

R6/R41/R43 add a single new boolean to `GridConfig` —
`coordinator_free_rounds` — and a single new counter to `GridMetrics` —
`coordinator_free_rounds: u32`. Defaults are tightly specified: the flag
defaults to `false` in v1 / non-delta mode (R43 SHOULD), so that NO v1
test, benchmark, or external caller changes behavior unless they opt in.
The metric counter starts at `0` and is incremented inside `run_grid`
(by TASK-0351) every time the coordinator skips merge.

This task only adds the data — no behavior changes. TASK-0351 wires
the consumption.

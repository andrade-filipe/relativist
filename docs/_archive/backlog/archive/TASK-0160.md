## Context

The current `src/io.rs` is a stub file with only a doc comment. SPEC-12 defines multiple subsystems (binary format, text DSL, JSON format, generators, summary) that must be organized into separate sub-modules. This task converts `src/io.rs` into a `src/io/` directory with a `mod.rs` that re-exports all public items. This mirrors the same pattern used in TASK-0001 for the `net` module and TASK-0080 for `protocol`.

The module layout follows SPEC-12 Section 4.3:

```
io/
  mod.rs          -- load_net, save_net, re-exports
  binary.rs       -- bincode serialization/deserialization
  text.rs         -- Text DSL parser and serializer
  json.rs         -- JSON serialization (stub in v1)
  examples.rs     -- Generator functions
  summary.rs      -- NetSummary, ReductionSummary
  types.rs        -- FileIoError, NetFormat
```

# TASK-0019: Implement get_agent and get_agent_mut accessors

**Spec:** SPEC-02
**Requirements:** R15a, R15b
**Priority:** P0 (critical path)
**Status:** TODO
**Depends on:** TASK-0008
**Blocked by:** none
**Estimated complexity:** S (< 50 LoC)

## Context

SPEC-02 Revised v3 (SC-001) added two new MUST-level requirements for agent accessors on `Net`. These replace the verbose inline pattern `self.agents.get(id as usize).and_then(|slot| slot.as_ref())` that was being duplicated across multiple specs (SPEC-14, SPEC-05, etc.) with a canonical public API.

`get_agent(id)` returns an immutable reference to the agent, and `get_agent_mut(id)` returns a mutable reference. Both are O(1) and return `None` if the ID is out of range or the agent slot is empty.

R15a declares `get_agent` as the canonical accessor: callers MUST NOT index into `agents` directly for read access. This centralizes bounds checking and the `Option` unwrapping pattern.

## Acceptance Criteria

- [ ] `get_agent(id: AgentId) -> Option<&Agent>` is a `pub` method on `Net`
- [ ] Returns `Some(&agent)` if `agents[id]` contains a live agent
- [ ] Returns `None` if `id` is out of range (`id as usize >= agents.len()`)
- [ ] Returns `None` if `agents[id]` is `None` (agent removed)
- [ ] `get_agent_mut(id: AgentId) -> Option<&mut Agent>` is a `pub` method on `Net`
- [ ] Returns `Some(&mut agent)` if `agents[id]` contains a live agent
- [ ] Returns `None` if `id` is out of range or the slot is empty
- [ ] Both operations are O(1)

## Key Types / Signatures

```rust
impl Net {
    /// Returns a reference to the agent with the given ID, or None if the
    /// ID is out of range or the slot is empty.
    ///
    /// This is the canonical accessor for agent lookup (R15a).
    /// Callers MUST NOT index into `agents` directly for read access.
    ///
    /// Complexity: O(1).
    pub fn get_agent(&self, id: AgentId) -> Option<&Agent> {
        self.agents.get(id as usize).and_then(|slot| slot.as_ref())
    }

    /// Returns a mutable reference to the agent with the given ID, or None
    /// if the ID is out of range or the slot is empty.
    ///
    /// Complexity: O(1).
    pub fn get_agent_mut(&mut self, id: AgentId) -> Option<&mut Agent> {
        self.agents.get_mut(id as usize).and_then(|slot| slot.as_mut())
    }
}
```

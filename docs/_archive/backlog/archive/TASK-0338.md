# TASK-0338: compute --encoder <name> --input <json> dispatch (R21, R23)

**Phase:** 5 (SPEC-27)
**Size:** S (~120 LoC inc. tests)
**Depends on:** TASK-0337 (default_registry)
**Spec:** SPEC-27 R21, R23

## Deliverable

Extend `ComputeArgs` (in `relativist-core/src/config.rs`) with two optional flags:

```rust
/// Encoder name (e.g., "lambda", "church_add"). When provided, --input is required
/// and the positional operation/a/b fields are ignored. SPEC-27 R21.
#[arg(long)]
pub encoder: Option<String>,

/// Encoder input as JSON string. Required when --encoder is provided. SPEC-27 R21.
#[arg(long, requires = "encoder")]
pub input: Option<String>,
```

Make the existing positional `operation`, `a`, `b` **optional** (so `compute --encoder lambda --input '...'` parses) and validate at runtime that EITHER:
- All three positional are present (legacy path), OR
- Both `--encoder` and `--input` are present (new path)

In `run_compute_command` (in `relativist-core/src/commands.rs`), branch on
`args.encoder.is_some()`:

- **Legacy path:** unchanged behaviour from SPEC-14.
- **New path:** use `default_registry()`:
  ```rust
  let registry = default_registry();
  let mut net = registry.encode_and_validate(name, input.as_bytes())?;  // R23 step 1+2
  reduce_all(&mut net);                                                  // R23 step 3
  let json_out = registry.decode(name, &net)?;                           // R23 step 4
  println!("{}", serde_json::to_string_pretty(&json_out)?);              // R23 step 5
  ```

Add `RegistryError` to `RelativistError` via `#[from]` so `?` works (or wrap manually).

- `relativist compute --encoder lambda --input '{"term":"(λx. x) (λy. y)"}'`
  prints a JSON object containing `"term"`.
- `relativist compute --encoder church_add --input '{"a":3,"b":5}'`
  prints `{"result":8}`.
- Missing `--input` when `--encoder` provided is rejected by clap.
- Unknown encoder name produces a clear error referencing the name.
- `cargo test --workspace` count: 777 → 781+ (≥ +4 unit tests for arg parsing
  + dispatch).
- Clippy + fmt clean.

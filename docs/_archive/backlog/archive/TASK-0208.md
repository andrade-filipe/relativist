## Context

This task implements the arithmetic correctness tests that require the reduction engine (`reduce_all` from SPEC-03). Each test builds an arithmetic net, reduces it to Normal Form, and verifies the result via `decode_nat`.

All test pseudocode uses the explicit two-step pattern from SC-004:
```rust
let mut net = build_op(a, b);
reduce_all(&mut net);
assert_eq!(decode_nat(&net), Some(expected));
```

ET-10 is a property test (proptest) with SHOULD priority.

## Acceptance Criteria

- [ ] ET-6: addition correctness for (a, b) in {(0,0), (0,1), (1,0), (1,1), (2,3), (10,20), (50,50), (100,100)}
- [ ] ET-7: multiplication correctness for (a, b) in {(0,1), (1,0), (1,1), (2,3), (5,5), (10,10)}
- [ ] ET-8: exponentiation correctness for (base, exp) in {(2,0), (2,1), (2,3), (2,8), (3,3)}
- [ ] ET-10 (SHOULD): proptest for random a, b in [0, 100] verifying addition correctness
- [ ] All tests use explicit two-step pattern: `reduce_all(&mut net)` then `decode_nat(&net)`
- [ ] No use of `reduce_all_and_return` (removed per SC-004)

## Key Types / Signatures

```rust
#[cfg(test)]
mod tests {
    use super::*;
    use crate::encoding::church::{encode_nat, decode_nat};
    use crate::reduction::reduce_all;

    #[test]
    fn et_6_addition_correctness() {
        for (a, b) in [(0,0), (0,1), (1,0), (1,1), (2,3), (10,20), (50,50), (100,100)] {
            let mut net = build_add(a, b);
            reduce_all(&mut net);
            assert_eq!(decode_nat(&net), Some(a + b), "add({a}, {b})");
        }
    }

    #[test]
    fn et_7_multiplication_correctness() {
        for (a, b) in [(0,1), (1,0), (1,1), (2,3), (5,5), (10,10)] {
            let mut net = build_mul(a, b);
            reduce_all(&mut net);
            assert_eq!(decode_nat(&net), Some(a * b), "mul({a}, {b})");
        }
    }

    #[test]
    fn et_8_exponentiation_correctness() {
        for (base, exp) in [(2,0), (2,1), (2,3), (2,8), (3,3)] {
            let mut net = build_exp(base, exp);
            reduce_all(&mut net);
            assert_eq!(decode_nat(&net), Some(base.pow(exp as u32)), "exp({base}, {exp})");
        }
    }
}
```

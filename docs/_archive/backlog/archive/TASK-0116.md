## Context

SPEC-07 Section 4.10 defines the `main` function: initialize tracing, parse CLI, dispatch to the correct subcommand handler, and map errors to exit codes. The function uses `#[tokio::main]` because coordinator and worker are async. Local mode and generate are synchronous but can run inside the tokio runtime via `spawn_blocking` or directly (since they don't use async I/O).

This is the final wiring task that connects all Phase 6 pieces into a working binary.

## Key Types / Signatures

```rust
use clap::Parser;
use relativist::config::{Cli, Command};
use relativist::observability::init_tracing;

#[tokio::main]
async fn main() {
    init_tracing();

    let cli = Cli::parse();

    let result = match cli.command {
        Command::Coordinator(args) => run_coordinator_command(args).await,
        Command::Worker(args) => run_worker_command(args).await,
        Command::Local(args) => {
            tokio::task::spawn_blocking(move || run_local_command(args))
                .await
                .unwrap_or_else(|e| Err(RelativistError::Config(format!("task panic: {}", e))))
        }
        Command::Reduce(args) => {
            tokio::task::spawn_blocking(move || run_reduce(&args))
                .await
                .unwrap_or_else(|e| Err(RelativistError::Config(format!("task panic: {}", e))))
        }
        Command::Inspect(args) => {
            tokio::task::spawn_blocking(move || run_inspect(&args))
                .await
                .unwrap_or_else(|e| Err(RelativistError::Config(format!("task panic: {}", e))))
        }
        Command::Generate(args) => {
            tokio::task::spawn_blocking(move || run_generate_command(args))
                .await
                .unwrap_or_else(|e| Err(RelativistError::Config(format!("task panic: {}", e))))
        }
        Command::Compute(args) => {
            tokio::task::spawn_blocking(move || run_compute(&args))
                .await
                .unwrap_or_else(|e| Err(RelativistError::Config(format!("task panic: {}", e))))
        }
    };

    match result {
        Ok(()) => std::process::exit(0),
        Err(e) => {
            tracing::error!("{}", e);
            std::process::exit(e.exit_code());
        }
    }
}
```

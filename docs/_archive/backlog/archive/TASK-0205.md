## Context

`build_mul(a: u64, b: u64) -> Net` constructs an IC net that, when reduced to Normal Form via `reduce_all`, yields `church(a * b)`. The function encodes `mul = lambda m. lambda n. lambda f. m (n f)`.

Multiplication involves function composition: church(a) applies the composed function `(church(b) f)` a total of `a` times. This creates O(a * b) interactions because DUP agents in church(a) duplicate the entire church(b) sub-net. This is the canonical Profile B benchmark for arithmetic (expansion + collapse cycle).

Per SPEC-14 Section 4.3.2, the construction uses 4 steps with explicit wire connections.

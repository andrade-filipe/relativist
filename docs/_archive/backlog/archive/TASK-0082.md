## Context

The `Message` enum is the central type of the wire protocol, defining all possible communication between coordinator and workers. It has 7 variants: 4 core protocol variants (`AssignPartition`, `Shutdown`, `PartitionResult`, `Error`) and 3 registration variants from SPEC-10 (`Register`, `RegisterAck`, `RegisterNack`). Every variant must be serializable via serde + bincode (R4).

> **SPEC-06 v3 change:** The enum was extended from 4 to 7 variants. Registration variants are included in the canonical SPEC-06 enum definition, with semantics owned by SPEC-10. R2a defines tier-dependent registration behavior. New variants MUST be appended at the end for bincode discriminant stability (R5). Note: TASK-0127 extends this enum with the registration variants -- the two tasks should be coordinated.

## Key Types / Signatures

```rust
use serde::{Serialize, Deserialize};

/// Messages in the communication protocol between coordinator and workers.
///
/// The enum covers all possible communication. Each variant is annotated
/// with the direction (Coordinator->Worker or Worker->Coordinator) and the
/// FSM state in which it is expected.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub enum Message {
    // === Coordinator -> Worker ===

    /// Sends a partition for the worker to reduce locally.
    /// Sent in the coordinator's Distributing state.
    AssignPartition {
        /// Current round number (0-indexed).
        round: u32,
        /// The complete partition to be reduced.
        partition: Partition,
    },

    /// Signals the worker to terminate.
    Shutdown,

    // === Worker -> Coordinator ===

    /// Returns the reduced partition to the coordinator.
    PartitionResult {
        /// Round number (echo of the value received in AssignPartition).
        round: u32,
        /// The partition with the locally-reduced sub-net.
        partition: Partition,
        /// Local reduction statistics for this worker in this round.
        stats: WorkerRoundStats,
    },

    /// Reports an irrecoverable error in the worker.
    Error {
        /// Round number in which the error occurred.
        round: u32,
        /// Identifier of the worker that reported the error.
        worker_id: WorkerId,
        /// Textual description of the error.
        description: String,
    },
}
```

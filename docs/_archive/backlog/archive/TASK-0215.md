## Context

SPEC-08 v3 (SC-008) identified that the invariant coverage matrix had no property-based tests for T2 (only principal ports form redexes) and T3 (active pairs are disjoint). Two new property tests were added:

- **PB13:** `prop_redex_detection_only_principal_ports` -- After building a random net, every entry in the redex queue is a pair connected via port 0. Directly tests T2 across random topologies.
- **PB14:** `prop_active_pairs_disjoint` -- After building a random net and running `reduce_all`, verify that no AgentId appears in more than one active pair in the redex queue at any step during reduction. Directly tests T3.

These tests use the `arb_net` proptest generator from `tests/common/generators.rs`.

```rust
use proptest::prelude::*;
use std::collections::HashSet;

proptest! {
    /// PB13: Every redex queue entry connects via principal ports (T2).
    #[test]
    fn prop_redex_detection_only_principal_ports(
        net in arb_net(50)
    ) {
        for &(a, b) in net.redex_queue() {
            // Both agents must exist
            prop_assert!(net.agent(a).is_some());
            prop_assert!(net.agent(b).is_some());
            // Connected via port 0
            prop_assert_eq!(net.get_target(PortRef::AgentPort(a, 0)),
                            PortRef::AgentPort(b, 0));
        }
    }

    /// PB14: No AgentId appears in more than one active pair (T3).
    #[test]
    fn prop_active_pairs_disjoint(
        net in arb_net(30)
    ) {
        let mut net = net;
        let budget = 100_000;
        for _ in 0..budget {
            // Check disjointness before each step
            let mut seen = HashSet::new();
            for &(a, b) in net.redex_queue() {
                if net.is_valid_redex(a, b) {
                    prop_assert!(seen.insert(a), "AgentId {} in multiple active pairs", a);
                    prop_assert!(seen.insert(b), "AgentId {} in multiple active pairs", b);
                }
            }
            if net.is_reduced() { break; }
            reduce_step(&mut net);
        }
    }
}
```

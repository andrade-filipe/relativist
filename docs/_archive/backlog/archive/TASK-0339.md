## Deliverable

Add a new `Encoders` variant to the `Command` enum in `relativist-core/src/config.rs`:

```rust
/// List / inspect registered encoders (SPEC-27 R22).
Encoders(EncodersArgs),
```

```rust
#[derive(clap::Args, Debug)]
pub struct EncodersArgs {
    #[command(subcommand)]
    pub action: EncodersAction,
}

#[derive(clap::Subcommand, Debug)]
pub enum EncodersAction {
    /// List all registered encoders with their descriptions.
    List,
}
```

Add dispatch handler `run_encoders_command` in `commands.rs`:

```rust
pub fn run_encoders_command(args: EncodersArgs) -> Result<(), RelativistError> {
    match args.action {
        EncodersAction::List => {
            let r = default_registry();
            println!("Available encoders:");
            let pairs = r.list();
            let max_name = pairs.iter().map(|(n, _)| n.len()).max().unwrap_or(0);
            for (name, desc) in pairs {
                println!("  {:<width$}  {}", name, desc, width = max_name);
            }
        }
    }
    Ok(())
}
```

Wire the dispatch in `main.rs` / `Command::Encoders(args) => run_encoders_command(args)`.

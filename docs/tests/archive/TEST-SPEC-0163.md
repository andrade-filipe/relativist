### E1: Empty bytes produce deserialization error

**Verify:** `deserialize_net(&[])` returns `Err(...)`, not a panic.
**Why:** An empty byte slice is invalid bincode; the error path must be exercised.

### E2: Truncated bytes produce deserialization error

**Verify:** Serialize a valid net, take only the first half of the bytes, attempt deserialization. Returns `Err(...)`.
**Why:** Partial data is a realistic failure mode (interrupted write, truncated file).

### E3: Large net roundtrip (1000 agents)

**Verify:** Create a net with 1000 ERA-ERA pairs (2000 agents, 1000 redexes), serialize and deserialize. All counts match.
**Why:** Validates that serialization scales to realistic benchmark sizes.
